# littlebot
slack bot (base on [botkit](https://github.com/howdyai/botkit))

## Installation
```
git clone https://github.com/snoopy2199/littlebot.git
npm install
```

## Getting Started
```
token={slackbot_token} node slack_littlebot.js
```
